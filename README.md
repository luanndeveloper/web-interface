# &#x20;Meu Primeiro Site - Projeto HTML

## 📌 Sobre o projeto

Este é um site desenvolvido como projeto do 1º bimestre do curso de **Informática para Internet** da Etec.🌐

O objetivo foi aplicar na prática os conhecimentos básicos de HTML, criando uma página pessoal com informações sobre mim.

---

## 👨‍💻 Tecnologias utilizadas

* HTML5

---

## 📚 Conteúdos aplicados

Neste projeto foram utilizados os seguintes conceitos de HTML:

* Estrutura básica do HTML (`<html>`, `<head>`, `<body>`)
* Títulos e parágrafos
* Links
* Imagens
* Listas
* Tabelas

---

## 🎯 Funcionalidades

O site contém:

* Informações pessoais
* Rotina diária
* Habilidades
* Tabela organizada com dados

---

## 🚀 Como visualizar o projeto

1. Baixe ou clone este repositório:

```
git clone https://github.com/seu-usuario/seu-repositorio.git
```

2. Abra o arquivo `index.html` no navegador
